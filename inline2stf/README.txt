The file 'ex1_inline.xml' and 'ex2_inline.xml' contain a text encoded using usual TEI inline markup.

The file 'inline2stf.xsl' is the XSLT stylesheet that transforms the document with inline markup to a document with standoff markup.

The file 'ex1_stf.xml' and 'ex2_stf.xml' are the standoff equivalents of 'ex1_inline.xml' and 'ex2_inline.xml', after the transformation using the XSLT stylesheet 'inline2stf.xsl'.

IMPORTANT: in order to obtain a valide document, a schema must be associated to the document containing the standoff markup.
The schema is in the folder 'schema'.
